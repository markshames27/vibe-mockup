# VirtualPainter - Clickable Prototype

**Homeowner Flow - Interactive Mockup**

This is a fully clickable, interactive prototype of the VirtualPainter widget for homeowners. Built with HTML5, Tailwind CSS, and Alpine.js.

---

## Quick Start

1. **Open the prototype**: Double-click `index.html` to open in your browser
2. **Click through the flow**: Navigate through all 9 screens
3. **Test interactions**: Try the slider, forms, buttons, and animations

---

## Screens Included

1. **Widget Landing** (`index.html`) - Introduction and CTA
2. **Upload Photo** (`upload.html`) - Choose camera or file upload
3. **Photo Preview** (`preview.html`) - Confirm photo quality
4. **Processing** (`processing.html`) - AI processing with loader
5. **Color Selection** (`color-selection.html`) - Choose from 12 colors
6. **Visualization Result** (`result.html`) - Before/after with slider
7. **Download/Share** (`download.html`) - Save or share visualization
8. **Contact Form** (`contact.html`) - Lead capture form
9. **Thank You** (`thankyou.html`) - Success confirmation

## How to Test

### Desktop Browser
1. Open `index.html` in Chrome, Firefox, Safari, or Edge
2. Click "Get Started - Upload Photo"
3. Follow the flow through all 9 screens
4. Test the interactive elements:
   - Hover effects on buttons
   - Before/after slider on result screen
   - Form validation on contact screen
   - Auto-redirect on processing screen

### Mobile Device
1. Transfer this folder to your phone
2. Open `index.html` in mobile browser
3. Experience the mobile-optimized design

### Developer Tools
1. Open browser DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Select iPhone or other mobile device
4. Test responsive behavior
